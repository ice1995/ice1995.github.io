解析库的安装

  1. lxml
    支持html和xml的解析，支持XPath解析方式，解析效率高

  2. Beautiful Soup

  3. pyquery
      pyquery 同样是一个强大的网页解析工具，它提供了和 jQuery 类似的语法来解析 HTML 文梢，支持 css 选择器，使用非常方便.

  4. tesserocr
      cor识别，识别各种各样的验证码。tesserocr是对tesseract的一层python封装，因此需要先安装tesseract
